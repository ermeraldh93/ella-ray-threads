# Ella Rae Threads Website v1

Static premium website for ellaraethreads.com.

## Deploy with GitHub + Cloudflare Pages
1. Create a GitHub repo named `ella-rae-threads`.
2. Upload all files from this folder.
3. Connect the repo to Cloudflare Pages.
4. Build command: leave blank.
5. Output folder: `/` or root.
6. Add the custom domain `ellaraethreads.com`.
